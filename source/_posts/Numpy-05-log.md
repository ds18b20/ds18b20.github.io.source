# Numpy.log

## numpy.log

Natural logarithm, **element-wise**.

```python
>>> np.log(np.e)
1.0
```

## numpy.log2

Base-2 logarithm of *x*.

```python
>>> np.log2(2)
1.0
```

## numpy.log10

Return the base 10 logarithm of the input array, element-wise.

```python
>>> np.log10(10)
1.0
```
